# Images Folder

This folder contains all images and assets for the project.

## Files:
- `logo.svg` - Main logo for the website
- `hero-bg.svg` - Hero section background
- `about-illustration.svg` - About section illustration
- `feature-1.svg` - Feature illustration

## Usage:
Import images in components using:
```javascript
<img src="/images/logo.svg" alt="Logo" />
```

## Adding New Images:
1. Place image files in this folder
2. Use descriptive names
3. Optimize images before adding
4. Update this README with new files
